# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Report vulnerabilities privately via GitHub's [Security Advisories](https://github.com/Python-Nigeria/pynigeria-backend/security/advisories/new) feature.

Please include:
- A description of the vulnerability and its potential impact
- Steps to reproduce or proof-of-concept
- Any suggested fix (optional)

You can expect:
- An acknowledgement within **48 hours**
- A status update within **7 days**
- Credit in the release notes once the issue is resolved (if desired)

We ask that you give us reasonable time to address the issue before any public disclosure.
